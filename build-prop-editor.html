Build Prop Editor





Note: This is just for educational purposes. Do not misuse.



Android

Some of the Android info given in About of the mobile can be changed. This Topic covers the way by which you can change your Android's Model Number and some other info.


Requirements

1. Build Prop Editor.apk

2. A Rooted Android Device (If you don't know how to root then click here).

3. An Active Internet Connection



Method

1. Open Google Chrome/ Play Store and download Build Prop Editor.apk or you can download it from any of the links given below:

‣ Link 1

‣ Link 2

‣ Link 3

2. Make sure that your android device is rooted. You can click here if you don't know how to root an Android Device.

3. Install and open Build Prop Editor (If mobile asks for permission while installing then permit it).

4. When you will open Build Prop Editor, grant it Supersu permission.

5. Once you have opened Build Prop Editor, you can change the following things in the following ways:

‣ Android version by locating ro.build.version.release and changing it to whatever you want.
‣ Android Model by locating ro.product.model and changing it to whatever you want.
‣ Product Brand by locating ro.product.brand and changing it whatever you want.

6. Congratulations, you are done. Enjoy!
