Android Lock Screen

3. Now, release just the Volume up Button and wait until the recovery screen appears and then release the other buttons also when the recovery screen appears.

4. Use volume keys to move to the wipe data factory reset option and then select it using the power button.

5. If it asks for confirmation, then select yes.

6. Now, your smartphone will reboot.
